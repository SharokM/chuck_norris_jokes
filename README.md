Chuck Norris Jokes

Chuck Norris Jokes is a simple React application that fetches and displays random jokes from a public API. Users can load a new joke by clicking a button.
This project is built as a React app to practice API fetching, React Hooks, and handling asynchronous data.

What This App Does
Fetches a random Chuck Norris joke from an external API
Displays the joke on the page
Allows users to fetch a new joke with a button click
Handles loading and error states

Technologies Used
React.js
React Hooks (useState, useEffect)
JavaScript
HTML5
CSS3

API Information
This project uses the public Chuck Norris Jokes API.

Endpoint used:
https://api.chucknorris.io/jokes/random

Each request returns a random joke as JSON.

How The App Works
When the app loads, a joke is fetched automatically
Clicking the button triggers a new API request
The fetched joke is stored in React state and rendered to the page
Errors are caught and logged to the console

Requirements
To run this project, you need:

Node.js (version 16 or higher)
npm (comes with Node.js)
A modern web browser

How To Run The App
Install dependencies
npm install

Start the app
npm start

Open in browser
http://localhost:3000

Notes
This is a React project
Uses a public API with no authentication
Internet connection required to fetch jokes
